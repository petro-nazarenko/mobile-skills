react-native-best-practices plugin

Privacy Policy

Effective date: April 10, 2026 • Plugin version: 1.0.0

Short version: This plugin collects no data. It contains only static Markdown skill files that run locally inside Claude Code on your machine. Nothing is transmitted to any external server by this plugin.

1. What this plugin is

react-native-best-practices is a Claude Code plugin distributed via the petro-nazarenko/mobile-skills marketplace. It consists of:

  • A SKILL.md file defining the /react-native-best-practices skill
  • Reference Markdown files (Metro, FlatList, Expo, Navigation, Platform)
  • A plugin.json manifest

There is no executable code, no server, no telemetry, and no network calls made by this plugin.

2. Data we collect

We collect no data of any kind. Specifically:

  • No personal information
  • No usage analytics or telemetry
  • No code or prompts from your projects
  • No crash reports or logs
  • No cookies or tracking identifiers

3. How the plugin operates

When you invoke /react-native-best-practices, Claude Code reads the local SKILL.md and reference files from your plugin cache (~/.claude/plugins/cache/) and passes their contents to the Claude model as part of your conversation context. This happens entirely within your Claude Code session under Anthropic's standard data handling — this plugin adds no additional data processing.

4. Third-party services

This plugin does not integrate with, call, or transmit data to any third-party service. Your use of Claude Code itself is governed by Anthropic's Privacy Policy.

5. GitHub hosting

The plugin source code is hosted on GitHub at petro-nazarenko/mobile-skills. Visiting that repository is subject to GitHub's Privacy Statement.

6. Children's privacy

This plugin is a developer tool not directed at children under 13. We do not knowingly collect information from children.

7. Changes to this policy

If this policy changes, the updated version will be published at this URL with a new effective date. Because no data is collected, no material changes are anticipated.
